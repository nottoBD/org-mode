* Keybindings

** Org Buffer (Notes Side)
| Key          | Action                                      |
|--------------|---------------------------------------------|
| SPC m n      | Start org-noter session                     |
| SPC m N      | Start org-noter session                     |
| RET / gf     | Open link at point (e.g., PDF file::page)  |
| C-c C-o      | Open link at point (built-in, vertical split)|
| M-p / M-n    | Previous/next note (sync PDF page)         |
| C-M-p / C-M-n| Previous/next precise note (jump exact location) |
| SPC m t l    | Toggle raw link display (for editing paths) |

** PDF Buffer (Document Side, Emacs State)
| Key          | Action                                      |
|--------------|---------------------------------------------|
| SPC m h      | Highlight selected text (pdf)               |
| i            | Insert approximate note (pdf)               |
| M-i          | Insert headline note (org)                                     |
| M-p / M-n    | Previous/next note (org)                    |
| q            | Quit session (close PDF, keep notes)        |
| k            | Kill session (close all buffers)            |
| s            | Sync current note to PDF location           |
| y            | Yank PDF location to Org (store-link)       |
| C-s / C-r    | Search forward/backward in PDF text (OCR/nav)|
| n / p        | Next/previous page                          |
| + / -        | Zoom in/out                                 |
| C-S-v        | Paste from clipboard (if needed in annot)   |
